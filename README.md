# Profitability Analysis Dashboard

## Tools Used
- Power BI
- Excel
- DAX

## Project Overview
This dashboard analyzes company profitability using key KPIs such as:
- Cost Ratio %
- Profit Margin %
- Total Sales
- Total Profit

## Key Insights
- North region generated highest profit (61M)
- Electronics category contributed highest profit (139M)
